# Meu portfólio [Em desenvolvimento]

Este é o repositório do meu portfólio pessoal, desenvolvido com React e TailwindCSS. Aqui você encontrará informações sobre mim, minhas habilidades, projetos e formas de contato.

## Tecnologias Utilizadas

- **React**: Biblioteca JavaScript para construção de interfaces de usuário.
- **TailwindCSS**: Framework CSS para estilização.
- **Vite**: Ferramenta de build rápida para desenvolvimento web.
- **ESLint**: Ferramenta de linting para manter a qualidade do código.

## Como Executar o Projeto

1. Clone o repositório:

```sh
git clone https://github.com/Leonnardo21/portfolio.git
```

2. Navegue até o diretório do projeto:

```sh
cd portfolio
```

3. Instale as dependências:

```sh
npm install
```

4. Execute o projeto em modo de desenvolvimento:

```sh
npm run dev
```

5. Abra o navegador e acesse <strong>http://localhost:port</strong>.

## Estrutura dos Componentes

Navbar: Barra de navegação com links para as seções do portfólio.
Main: Seção principal com uma breve introdução.
About: Seção sobre mim, com uma foto e uma descrição.
Skills: Seção que lista minhas habilidades e tecnologias que utilizo.
Projects: Seção que apresenta meus projetos com descrições e tecnologias utilizadas.
Contact: Seção com informações de contato.
Footer: Rodapé com direitos autorais.
